# Resume Database Management System

## 📌 Project Overview

Resume Database Management System is a Java Swing-based application that allows users to upload resumes, extract information from PDF files, store the extracted data in a structured format, and generate reports.

The project demonstrates the use of:

* Java Swing GUI
* File Handling
* Apache PDFBox
* JTable for Data Display
* Report Generation
* Event Handling

This project was developed as part of an academic assignment to read data from files and display it in a well-structured format using Swing and Applet concepts.

---

## 🚀 Features

* Upload Resume PDF
* Extract Resume Information
* Display Data in JTable
* Generate Resume Reports
* User-Friendly GUI
* Record Count Display
* PDF Parsing using Apache PDFBox

---

## ⚙️ How to Run

1. Open the project in Eclipse IDE.
2. Add Apache PDFBox JAR to the Build Path.
3. Run LoginPage.java or StudentReportViewer.java.
4. Click "Upload Resume PDF".
5. Select a PDF resume.
6. View extracted information in the table.
7. Click "Generate Report" to generate a report.
